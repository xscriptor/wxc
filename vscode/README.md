<h1 align="center">Previews for VS Code</h1>

These are preview images of themes and the XGlass extension.

This is only a preview. For full documentation, complete source code, and theme details, visit the repository:

- <a href="https://github.com/xscriptor/vscode">https://github.com/xscriptor/vscode</a>
